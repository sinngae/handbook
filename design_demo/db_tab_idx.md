199取余 均分证明
---
# 分表
0*10^n ~ 198*10^n 共199个数字

假设存在
```math
 0 <= x < y <= 198
 (x*10^n + d) % 199 = m
 (y*10^n + d) % 199 = m
```
=>
```
 a * 199 + m = x*10^n + d
 b * 199 + m = y*10^n + d
 且 0 <= a < b <= 198
```
=> 两等式相减
```
 (b-a)*199 = （y-x)*10^n
```
10^n的因子只有1，2，5，也即 y-x 必是 199 倍数,
而 0 < y - x < y < 198

=> 上述不存在
=> sloId 在 199 均匀分布

# 分库后
0*10^n ~ 1989*10^n 共 1990 个数字

假设存在
```
 0 <= x < y <= 1989
 (x*10^n + d) % 1990 = m
 (y*10^n + d) % 1990 = m
```
=>
```
 a * 1990 + m = x*10^n + d
 b * 1990 + m = y*10^n + d
 且 0 <= a < b <= 1989
```
=> 两等式相减
```
 (b-a)*1990 = （y-x)*10^n
 ```
10^n的因子只有1，2，5，也即 y-x 必是 199 倍数,
而 0 < y - x < y < 198

=> 上述存在 199 398 等等

=> sloId 使用 1990取模 不能均匀分布

=> sloId 应该使用 (sloId / 10^n) % 1990 % 10 取库idx，与(sloId / 10^n) % 10相同； (sloId / 10^n) % 1990 / 10 取表idx；

		或 (sloId / 10^n) % 1990 / 199 取库idx； (sloId / 10^n) % 1990 % 199 取表idx，与(sloId / 10^n) % 199相同；
