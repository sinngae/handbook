# Distributed system
网络解决了计算机之间的通讯，而分布式系统要解决的是多个计算机（或虚拟计算机）更高层次的协作。

分布的多个计算机，节点自治，任务分解并行在各节点，共同提供统一的服务。

难点：
1.分区故障导致的系统瘫痪
2.性能和可靠性依赖网络
3.统一控制，机制与策略，动态扩容/缩容
4.资源的利用率低
5.安全，服务安全/数据安全

+ 仅传输一些必要的网络数据，降低网络流量
